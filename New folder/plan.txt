**Посты пользователя:**

- vkUserid - Внутренний id вк Владельца страницы (используется только для проверки точности результата участника)
owner_id()

- vkPostID - Внутренний id поста вк (используется только для проверки точности результата участника)
id()

- date - Дата поста
date()

- text - Текст поста
text()

- author - Автор поста (id)
from_id()

- likesCount - Количество лайков на посте.
likes.count()

- friend - true\false от друга ли пост?
сравнить автора со списком друзей, если автор не владелец страницы
